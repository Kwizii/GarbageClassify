## 功能模块集成：
- 用户管理：登录、注册..
- 垃圾识别AI分类：通过Python部署模型并通过Flask开放接口供远程调用..
- 知识问答：在线答题、考试管理、题库管理..
- 积分商城：积分置换、历史订单..
